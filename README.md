🐾 Veterinaria Terán

Sitio web desarrollado para Veterinaria Terán, una empresa con múltiples locales dedicada al cuidado integral de mascotas 🐶🐱.

Este proyecto fue construido utilizando React, enfocándose en una interfaz moderna, responsive y fácil de usar para los clientes.

🚀 Descripción

La plataforma web permite a los usuarios:

Conocer los servicios ofrecidos 🏥
Visualizar los diferentes locales 📍
Contactarse fácilmente con la veterinaria 📞
Explorar contenido visual (galería, equipo, etc.) 🖼️

El objetivo principal es brindar una experiencia digital clara, atractiva y accesible para los clientes.

🛠️ Tecnologías utilizadas
⚛️ React
🎨 CSS / Diseño Responsive
📦 Node.js & npm
